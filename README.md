# card_roguelike

## 介绍

仿《杀戮尖塔》的卡牌构筑`rogue like`游戏。

## 战斗流程

![战斗流程](./docs/procedure_combat.png)

## 计划内容

- 基于卡牌的回合战斗系统
  - 牌组的随机和管理
  - 卡牌的CSV配置，数据驱动
  - 抽牌、出牌、弃牌等基础操作
- 可扩展的AbilitySystem技能系统
  - 抽象BUFF
  - 属性管理`Attribute`
- 可预测的怪物AI（基于行为树）
